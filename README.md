# Trades Today — Issue Tracker

Welcome to the public issue tracker for **Trades Today**, a trading copilot app designed to help traders make smarter, faster decisions.

> **Note:** This repository is dedicated to issue tracking only. The application source code is maintained in a separate private repository.

---

## About Trades Today

Trades Today is a trading copilot that assists traders with real-time insights, trade management, and decision support. The app includes several key sections to support your trading workflow.

---

## How to Report an Issue

We use GitHub Issues to track bugs, feature requests, and general feedback. Before opening a new issue, please:

1. **Search existing issues** to avoid duplicates.
2. **Use the appropriate issue template** when creating a new issue.
3. **Provide as much detail as possible** — screenshots, steps to reproduce, and expected vs. actual behavior are all helpful.

### Issue Templates

| Template | Use When |
|---|---|
| Bug Report | Something is broken or not working as expected |
| Feature Request | You have an idea for a new feature or improvement |
| UI / UX Feedback | Feedback on the app's design, layout, or usability |
| Data / Accuracy Issue | Trade data, pricing, or analytics appear incorrect |
| General Question | You have a question about how something works |

---

## Labels

Issues are organized with labels to help prioritize and categorize:

| Label | Description |
|---|---|
| `bug` | Something isn't working |
| `enhancement` | New feature or request |
| `ui/ux` | Related to design and user experience |
| `data-accuracy` | Issues with trade data or calculations |
| `question` | Further information is requested |
| `good first issue` | Good for newcomers |
| `priority: high` | Needs immediate attention |
| `priority: medium` | Should be addressed soon |
| `priority: low` | Nice to have, not urgent |
| `in-progress` | Currently being worked on |
| `wontfix` | This will not be worked on |
